The Echo ATS is the next-generation successor to the modular Echo AIO platform,
integrating the most-used measurement capabilities into a single compact
interface. This page provides a side-by-side hardware comparison to help you
evaluate the two systems.

<Note>
The AIO is a modular system; no single AIO unit provides every capability at
once. The AIO column below reflects the best case across the current acoustic
and analog modules (A, L, S, and H). The Combo (C) and TDM (T) modules are not
included.
</Note>

## Summary

Across the core measurement engine — input and output fidelity, channel
density, gain flexibility, and footprint — the ATS **meets or exceeds** the
AIO. The ATS also adds **SPDIF / Word Clock digital I/O** and **GPIO** that the
comparable AIO configuration does not provide.

The AIO retains an edge in only three areas, none of which affect audio
fidelity:

1. **Maximum line-output voltage** — 16 Vrms balanced vs. the ATS's 8 Vrms.
2. **Sustained amplifier power** — rated 10 W continuous vs. the ATS's 10 W
   peak / 5 W continuous.
3. **Line-output count** — a dual-module AIO-L2 provides 4 line outputs vs. the
   ATS's 2.

## Side-by-Side Comparison

| Spec | Echo ATS | Echo AIO (A / L / S / H) | Advantage |
|---|---|---|---|
| Mic/Line inputs | 8 | up to 8 | Tie |
| Input dynamic range | 115 dB | 112 dB | **ATS** |
| Input THD+N | −110 dB | < −105 dB | **ATS** |
| Input gain | 5 steps, 0 → 40 dB (10 dB) | 3 steps (1× / 10× / 100×) | **ATS** |
| Input coupling | Switchable AC/DC | AC only | **ATS** |
| IEPE / CCP | 4 mA @ 24 V | 4 mA | **ATS** |
| TEDS | Yes | Yes (IEEE 1451.4 Cl. 1) | Tie |
| Line outputs | 2 | up to 4 (L2) | **AIO** (count) |
| Line out max voltage | 8 Vrms bal | 16 Vrms bal | **AIO** |
| Line out dynamic range | 120 dB | 120 dB | Tie |
| Line out THD+N | −113 dB | < −102 dB | **ATS** |
| Amplifier outputs | 4 | up to 4 | Tie |
| Amplifier power | 10 W peak / 5 W cont. | 10 W cont. (8 Ω) | **AIO** (sustained) |
| Amplifier dynamic range | 108 dB (A-wtd) | 100 dB | **ATS** |
| Headphone output | 3 Vrms, ≥ 16 Ω | 3 Vrms, ≥ 16 Ω | Tie |
| Digital I/O | SPDIF / Word Clock | None | **ATS** |
| GPIO | 4 in / 4 out | None | **ATS** |
| Size | 11 × 2 × 8.5 in | 17.5 × 8.75 × 1.75 in (1U) | **ATS** |
| Power | 40 W | 60 W | **ATS** |

<Note>
The AIO datasheet publishes impedance-measurement accuracy figures
(≤ 0.5% speaker, ≤ 1% headphone). The ATS performs impedance measurement via
0.1 Ω current sensing; a published accuracy figure is forthcoming.
</Note>

## Bottom Line

Excluding the deprecated Combo and TDM modules, the ATS matches or exceeds the
AIO on every measurement-quality and connectivity specification, and adds
digital I/O and GPIO that the comparable AIO lacks. The only specifications
where the AIO still leads are higher maximum line-output voltage, higher
*sustained* amplifier power, and more line outputs in a dual-module
configuration — none of which involve audio fidelity.
