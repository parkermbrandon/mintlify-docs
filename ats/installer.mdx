The Echo Test Interfaces installer includes:

- **ASIO/Core Audio Drivers** - Low-latency audio drivers for Windows and macOS
- **ATS Control Panel** - GUI application for device configuration and audio routing
- **EchoCLI** - Command-line interface for scripting and automation
- **EchoAPI** - Library for custom application development
