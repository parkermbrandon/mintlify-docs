## Update Instructions

### Windows Firmware Update

1. Connect your ATS system via USB
2. Open the ATS Control Panel
3. Navigate to the Options tab
4. Click "Update Firmware"
5. Select the downloaded firmware file
6. Wait for the update to complete (approximately 30 seconds)
7. The system will automatically restart
