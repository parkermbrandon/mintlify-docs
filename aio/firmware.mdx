## Installation Instructions

### First-Time Installation

1. Download the AIO Software Suite installer above
2. Close all other applications
3. Run the installer as Administrator (right-click → "Run as administrator")
4. Follow the installation wizard:
   - Accept the license agreement
   - Choose installation directory (default recommended)
   - Select components to install
5. Connect your AIO device when prompted
6. The installer will automatically install USB drivers
7. Restart your computer when installation completes

### Updating Existing Installation

1. Download the latest installer
2. The installer will detect your existing installation
3. Select "Update" when prompted
4. Your settings and calibration data will be preserved
5. Restart the AIO software after update completes

## Firmware Updates

The AIO firmware is included with the software installer and can be updated through the AIO Control Software.

### How to Update Firmware

1. Launch the AIO Control Software
2. Connect your AIO device via USB
3. Navigate to **Settings → Firmware**
4. Click "Check for Updates"
5. If an update is available, click "Update Firmware"
6. **Do not disconnect** the device during update (typically 30-60 seconds)
7. The device will restart automatically when complete

### Manual Firmware Update

If automatic update fails, you can update manually:

1. Download the firmware file from the software installation directory:
   `C:\Program Files\Echo Test & Measurement\AIO\Firmware\`
2. In AIO Control Software, go to **Settings → Firmware**
3. Click "Manual Update"
4. Select the firmware file
5. Follow the on-screen instructions

## Troubleshooting

<AccordionGroup>
  <Accordion title="Installer won't run">
    - Ensure you have Administrator privileges
    - Temporarily disable antivirus software
    - Check that you have sufficient disk space
    - Try downloading the installer again
  </Accordion>

  <Accordion title="USB drivers not installing">
    - Disconnect and reconnect the AIO device
    - Try a different USB port
    - Manually install drivers from:
      `C:\Program Files\Echo Test & Measurement\AIO\Drivers\`
    - Check Device Manager for any error indicators
  </Accordion>

  <Accordion title="Firmware update fails">
    - Ensure stable power connection
    - Try a different USB cable
    - Close all other applications
    - Use manual firmware update method
    - Contact support if issue persists
  </Accordion>
</AccordionGroup>

## System Requirements

### Minimum Requirements
- **OS:** Windows 10 version 1903 or later (64-bit)
- **Processor:** Intel Core i3 or AMD equivalent
- **RAM:** 4GB
- **Storage:** 500MB available space
- **USB:** USB 2.0 port
- **Display:** 1280x720 resolution

### Recommended Requirements
- **OS:** Windows 11 (64-bit)
- **Processor:** Intel Core i5 or AMD equivalent
- **RAM:** 8GB or more
- **Storage:** 1GB available space
- **USB:** USB 3.0 port or higher
